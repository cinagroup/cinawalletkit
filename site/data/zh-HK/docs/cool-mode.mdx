# 酷炫模式

## 啟用酷炫模式 🌈🌈🌈

您可以在 `CinaWalletKitProvider` 上使用 `coolMode` 屬性來啟用酷炫模式。

```tsx
import { CinaWalletKitProvider } from '@cinagroup/CinaWalletKit';

const App = () => (
  <CinaWalletKitProvider coolMode {...etc}>
    {/* Your App */}
  </CinaWalletKitProvider>
);
```

### 但這是什麼？

選擇錢包列表中的錢包時出現的表情符號爆炸 🌈🌈🌈。它會使用您選擇的錢包圖示。
