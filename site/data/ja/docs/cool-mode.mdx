# クールモード

## クールモードの有効化 🌈🌈🌈

`CinaWalletKitProvider`の`coolMode`プロップを使ってクールモードを有効にすることができます。

```tsx
import { CinaWalletKitProvider } from '@cinagroup/CinaWalletKit';

const App = () => (
  <CinaWalletKitProvider coolMode {...etc}>
    {/* Your App */}
  </CinaWalletKitProvider>
);
```

### でも、それは何ですか？

ウォレットリストからウォレットを選択すると、絵文字の爆発が🌈🌈🌈表示されます。選択したウォレットのアイコンを使用します。
