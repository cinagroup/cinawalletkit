# カスタムウォレットリスト

## ウォレットリストのカスタマイズ

必要なコネクタを持つウォレットのリストを構築するために、`'@cinagroup/CinaWalletKit/wallets'` から個別のウォレットと `connectorsForWallets` 関数をインポートできます。この方法で、どのウォレットをどの順番で表示するかを完全に制御できます。

例えば、Rainbow と一般的なフォールバックウォレットのみを表示することができます。

```tsx
import { connectorsForWallets } from '@cinagroup/CinaWalletKit';
import {
  rainbowWallet,
  walletConnectWallet,
} from '@cinagroup/CinaWalletKit/wallets';

const connectors = connectorsForWallets(
  [
    {
      groupName: 'Recommended',
      wallets: [rainbowWallet, walletConnectWallet],
    },
  ],
  {
    appName: 'My CinaWalletKit App',
    projectId: 'YOUR_PROJECT_ID',
  }
);
```

その後、Wagmi の `createConfig` にコネクタを渡すことができます。

```tsx
import { connectorsForWallets } from '@cinagroup/CinaWalletKit';
import { createConfig } from 'wagmi';

const connectors = connectorsForWallets(/* ... */);

const config = createConfig({
  connectors,
  {/* Wagmi config */}
});

const queryClient = new QueryClient();

const App = () => (
  <WagmiProvider config={config}>
    <QueryClientProvider client={queryClient}>
      <CinaWalletKitProvider {...etc}>
        {/* Your App */}
      </CinaWalletKitProvider>
    </QueryClientProvider>
  </WagmiProvider>
);
```

### 内蔵ウォレット

以下のウォレットは `wallet` オブジェクトを介して提供されます。

#### 一般ウォレット

以下のウォレットは特定のアプリではなく、一般的な接続方法に対応しています。そのため、これらのウォレットは常に含めることをお勧めします。

##### ウォレットコネクト

これは、他のウォレットリストに提供されていないウォレットコネクトベースのウォレットのために設計されたフォールバックウォレットオプションです。

**ウォレットコネクトベースのすべてのウォレットがサポートされるように、常にこのウォレットをリストに含めることをお勧めします。**

```tsx
import { walletConnectWallet } from '@cinagroup/CinaWalletKit/wallets';
```

##### インジェクテッドウォレット

これは、`window.ethereum` が存在するが他のウォレットリストに提供されていないシナリオのために設計されたフォールバックウォレットオプションです。このウォレットは、フォールバックが必要でないか、インジェクテッドウォレットが利用できない場合にはリストから自動的に非表示になります。

**すべてのインジェクテッドウォレットがサポートされるように、常にこのウォレットをリストに含めることをお勧めします。**

```tsx
import { injectedWallet } from '@cinagroup/CinaWalletKit/wallets';
```

##### セーフウォレット

SafeアプリをサポートするdAppsには、ユーザーがSafeボールトやマルチシグに簡単に接続できるように`safeWallet`オプションを含めてください。SafeオプションはSafeウォレットブラウザ環境にのみ表示されます。

```tsx
import { safeWallet } from '@cinagroup/CinaWalletKit/wallets';
```

#### ベース（旧コインベースウォレット）

このウォレットコネクタは、コインベーススマートウォレットの進化であるベースをサポートします。サブアカウントや支出許可といった強化された機能とシームレスな認証を提供します。

構成オプションは[Base SDKのリファレンス](https://docs.base.org/base-account/reference/core/createBaseAccount)に準拠しています。

> `baseAccount`および`coinbaseWallet`は、後方互換のエイリアスとして引き続き利用可能です。

```tsx
import { base } from '@cinagroup/CinaWalletKit/wallets';
import { base as baseChain } from 'wagmi/chains';

// Configure attribution (choose one approach)
base.preference = {
  attribution: { auto: true }, // Smart Wallet generates origin-based identifier
};

// Or use custom attribution
base.preference = {
  attribution: { dataSuffix: '0x1234567890abcdef' }, // 16-byte hex identifier
};

// Configure sub-accounts
base.subAccounts = {
  creation: 'on-connect', // or 'manual'
  defaultAccount: 'sub', // or 'universal'
  funding: 'spend-permissions', // or 'manual'
};

// Configure paymaster URLs for gasless transactions
base.paymasterUrls = {
  [baseChain.id]: 'https://api.developer.coinbase.com/rpc/v1/base/...',
};
```

### 例

#### 順序

異なるウォレットを異なる順序で表示する例をいくつか示します。

MetaMask と一般的なフォールバックウォレットを表示します。

```tsx
import { connectorsForWallets } from '@cinagroup/CinaWalletKit';
import {
  metaMaskWallet,
  walletConnectWallet,
} from '@cinagroup/CinaWalletKit/wallets';

const connectors = connectorsForWallets(
  [
    {
      groupName: 'Recommended',
      wallets: [metaMaskWallet, walletConnectWallet],
    },
  ],
  { appName: 'CinaWalletKit App', projectId: 'YOUR_PROJECT_ID' },
);
```

Rainbow、MetaMask、Baseおよび一般的なフォールバックウォレットを表示します。

```tsx
import { connectorsForWallets } from '@cinagroup/CinaWalletKit';
import {
  rainbowWallet,
  metaMaskWallet,
  base,
  walletConnectWallet,
} from '@cinagroup/CinaWalletKit/wallets';

const connectors = connectorsForWallets(
  [
    {
      groupName: 'Suggested',
      wallets: [
        rainbowWallet,
        metaMaskWallet,
        base,
        walletConnectWallet,
      ],
    },
  ],
  { appName: 'CinaWalletKit App', projectId: 'YOUR_PROJECT_ID' },
);
```

> 注：`wallets` 配列の順序は、UI に表示されるウォレットの順序を決定します。

#### 複数のグループ

`groupName` キーを使用して、異なるウォレットグループに名前を付けることができます。これにより、ユーザーに推奨するウォレットや他の可能なウォレットを伝えることができます。

RainbowとMetaMaskを推奨しますが、Baseと一般的なフォールバックウォレットも提供します。

```tsx
import { connectorsForWallets } from '@cinagroup/CinaWalletKit';
import {
  rainbowWallet,
  metaMaskWallet,
  base,
  walletConnectWallet,
} from '@cinagroup/CinaWalletKit/wallets';

const connectors = connectorsForWallets(
  [
    {
      groupName: 'Recommended',
      wallets: [rainbowWallet, metaMaskWallet],
    },
    {
      groupName: 'Others',
      wallets: [base, walletConnectWallet],
    },
  ],
  { appName: 'CinaWalletKit App', projectId: 'YOUR_PROJECT_ID' },
);
```
