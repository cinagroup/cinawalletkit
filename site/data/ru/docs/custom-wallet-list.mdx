# Пользовательский список кошельков

## Настройка списка кошельков

Вы можете импортировать индивидуальные кошельки из `'@cinagroup/CinaWalletKit/wallets'` вместе с функцией `connectorsForWallets`, чтобы создать свой собственный список кошельков с их необходимыми соединителями. Таким образом, у вас будет полный контроль над тем, какие кошельки отображать и в каком порядке.

Например, вы можете выбрать отображение только Rainbow вместе с универсальными резервными кошельками.

```tsx
import { connectorsForWallets } from '@cinagroup/CinaWalletKit';
import {
  rainbowWallet,
  walletConnectWallet,
} from '@cinagroup/CinaWalletKit/wallets';

const connectors = connectorsForWallets(
  [
    {
      groupName: 'Recommended',
      wallets: [rainbowWallet, walletConnectWallet],
    },
  ],
  {
    appName: 'My CinaWalletKit App',
    projectId: 'YOUR_PROJECT_ID',
  }
);
```

Вы можете затем передать свои соединители в Wagmi `createConfig`.

```tsx
import { connectorsForWallets } from '@cinagroup/CinaWalletKit';
import { createConfig } from 'wagmi';

const connectors = connectorsForWallets(/* ... */);

const config = createConfig({
  connectors,
  {/* Wagmi config */}
});

const queryClient = new QueryClient();

const App = () => (
  <WagmiProvider config={config}>
    <QueryClientProvider client={queryClient}>
      <CinaWalletKitProvider {...etc}>
        {/* Your App */}
      </CinaWalletKitProvider>
    </QueryClientProvider>
  </WagmiProvider>
);
```

### Встроенные кошельки

Следующие кошельки предоставляются через объект `wallet`.

#### Универсальные кошельки

Следующие кошельки ориентированы на универсальные методы подключения, а не на конкретные приложения. В результате рекомендуется всегда включать эти кошельки.

##### WalletConnect

Это резервный вариант кошелька, предназначенный для кошельков на базе WalletConnect, которые не были предоставлены другим кошельком в списке.

**Рекомендуется всегда включать этот кошелек в список, чтобы обеспечить поддержку всех кошельков на базе WalletConnect.**

```tsx
import { walletConnectWallet } from '@cinagroup/CinaWalletKit/wallets';
```

##### Встроенный кошелек

Это резервный вариант кошелька, предназначенный для ситуаций, когда `window.ethereum` существует, но не был предоставлен другим кошельком в списке. Этот кошелек автоматически скроет себя из списка, когда резерв не требуется или если доступного встроенного кошелька нет.

**Рекомендуется всегда включать этот кошелек в список, чтобы обеспечить поддержку всех встроенных кошельков.**

```tsx
import { injectedWallet } from '@cinagroup/CinaWalletKit/wallets';
```

##### Безопасный кошелек

Для dApps, которые поддерживают Safe Apps, предоставляя пользователям возможность легко подключаться к их Safe vault или мульти-сиг, включите опцию `safeWallet` для автоматического подключения. Опция Safe будет отображаться только в браузерной среде Safe Wallet.

```tsx
import { safeWallet } from '@cinagroup/CinaWalletKit/wallets';
```

#### База (ранее Coinbase Wallet)

Этот коннектор кошелька поддерживает Base, эволюцию Coinbase Smart Wallet. Он обеспечивает бесшовную аутентификацию и расширенные функции, такие как под-аккаунты и разрешения на траты.

Параметры конфигурации соответствуют [справочнику Base SDK](https://docs.base.org/base-account/reference/core/createBaseAccount).

> `baseAccount` и `coinbaseWallet` остаются доступными как обратно-совместимые псевдонимы.

```tsx
import { base } from '@cinagroup/CinaWalletKit/wallets';
import { base as baseChain } from 'wagmi/chains';

// Configure attribution (choose one approach)
base.preference = {
  attribution: { auto: true }, // Smart Wallet generates origin-based identifier
};

// Or use custom attribution
base.preference = {
  attribution: { dataSuffix: '0x1234567890abcdef' }, // 16-byte hex identifier
};

// Configure sub-accounts
base.subAccounts = {
  creation: 'on-connect', // or 'manual'
  defaultAccount: 'sub', // or 'universal'
  funding: 'spend-permissions', // or 'manual'
};

// Configure paymaster URLs for gasless transactions
base.paymasterUrls = {
  [baseChain.id]: 'https://api.developer.coinbase.com/rpc/v1/base/...',
};
```

#### Кошелек Gemini

Этот умный коннектор кошелька позволяет безупречно аутентифицироваться через ключи на keys.gemini.com — без загрузок, расширений или API ключей. Просто работает из коробки.

Параметр `appName` необходим для идентификации вашего dApp, а `appIcon` является необязательным для брендинга опыта подключения.

```tsx
import { geminiWallet } from '@cinagroup/CinaWalletKit/wallets';

const gemini = geminiWallet({
  appName: 'My App',
  appIcon: 'https://my-app.com/logo.png', // Optional
});
```

### Примеры

#### Заказ

Вот несколько примеров отображения различных кошельков в разном порядке.

Покажите MetaMask вместе с универсальными резервными кошельками.

```tsx
import { connectorsForWallets } from '@cinagroup/CinaWalletKit';
import {
  metaMaskWallet,
  walletConnectWallet,
} from '@cinagroup/CinaWalletKit/wallets';

const connectors = connectorsForWallets(
  [
    {
      groupName: 'Recommended',
      wallets: [metaMaskWallet, walletConnectWallet],
    },
  ],
  { appName: 'CinaWalletKit App', projectId: 'YOUR_PROJECT_ID' },
);
```

Показать Rainbow, MetaMask и Base вместе с общими резервными кошельками.

```tsx
import { connectorsForWallets } from '@cinagroup/CinaWalletKit';
import {
  rainbowWallet,
  metaMaskWallet,
  base,
  walletConnectWallet,
} from '@cinagroup/CinaWalletKit/wallets';

const connectors = connectorsForWallets(
  [
    {
      groupName: 'Suggested',
      wallets: [
        rainbowWallet,
        metaMaskWallet,
        base,
        walletConnectWallet,
      ],
    },
  ],
  { appName: 'CinaWalletKit App', projectId: 'YOUR_PROJECT_ID' },
);
```

> Напоминание: порядок массива `кошельков` определяет порядок, в котором кошельки будут отображаться в интерфейсе.

#### Несколько групп

Вы можете использовать ключ `groupName`, чтобы назвать различные группы кошельков. Это удобно, если вы хотите сообщить пользователям, какие кошельки вы рекомендуете, а также другие возможные кошельки.

Рекомендовать Rainbow и MetaMask, но также предлагать Base вместе с общими резервными кошельками.

```tsx
import { connectorsForWallets } from '@cinagroup/CinaWalletKit';
import {
  rainbowWallet,
  metaMaskWallet,
  base,
  walletConnectWallet,
} from '@cinagroup/CinaWalletKit/wallets';

const connectors = connectorsForWallets(
  [
    {
      groupName: 'Recommended',
      wallets: [rainbowWallet, metaMaskWallet],
    },
    {
      groupName: 'Others',
      wallets: [base, walletConnectWallet],
    },
  ],
  { appName: 'CinaWalletKit App', projectId: 'YOUR_PROJECT_ID' },
);
```
