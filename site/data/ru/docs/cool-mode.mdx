# Крутой режим

## Включение крутого режима 🌈🌈🌈

Вы можете включить крутой режим через свойство `coolMode` в `CinaWalletKitProvider`.

```tsx
import { CinaWalletKitProvider } from '@cinagroup/CinaWalletKit';

const App = () => (
  <CinaWalletKitProvider coolMode {...etc}>
    {/* Your App */}
  </CinaWalletKitProvider>
);
```

### Но что это такое?

Эмоции взрываются, когда выбирается кошелек из списка кошельков 🌈🌈🌈. Используется значок выбранного кошелька.
