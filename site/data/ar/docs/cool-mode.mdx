# الوضع الرائع

## تمكين الوضع الرائع 🌈🌈🌈

يمكنك تمكين الوضع الرائع عبر الخاصية `coolMode` في `CinaWalletKitProvider`.

```tsx
import { CinaWalletKitProvider } from '@cinagroup/CinaWalletKit';

const App = () => (
  <CinaWalletKitProvider coolMode {...etc}>
    {/* Your App */}
  </CinaWalletKitProvider>
);
```

### لكن ما هو؟

انفجار الرموز التعبيرية عند اختيار محفظة من قائمة المحافظ 🌈🌈🌈. يتم استخدام أيقونة المحفظة التي تختارها.
