# أحجام النوافذ المنبثقة

## مقارنة الحجم الواسع والمضغوط

افتراضيًا، يتم تعيين حجم نافذة CinaWalletKit إلى `واسع` (ويبدو هكذا):

<Img src="/modal-default.png" height="569" width="1029" />

كما نوفر أيضًا حجم نافذة `مضغوط`، والذي يبدو هكذا:

<Img src="/modal-compact.png" height="569" width="1029" />

لتعيين حجم نافذة الخاص بك إلى `مضغوط`، ما عليك سوى إضافة `modalSize="compact"` كخاصية في `CinaWalletKitProvider`:

```tsx
import { CinaWalletKitProvider } from '@cinagroup/CinaWalletKit';

export const App = () => (
  <CinaWalletKitProvider modalSize="compact" {...etc}>
    {/* Your App */}
  </CinaWalletKitProvider>
);
```
