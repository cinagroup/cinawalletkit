# Chế độ làm mát

## Kích hoạt chế độ làm mát 🌈🌈🌈

Bạn có thể kích hoạt chế độ làm mát qua thuộc tính `coolMode` trên `CinaWalletKitProvider`.

```tsx
import { CinaWalletKitProvider } from '@cinagroup/CinaWalletKit';

const App = () => (
  <CinaWalletKitProvider coolMode {...etc}>
    {/* Your App */}
  </CinaWalletKitProvider>
);
```

### Nhưng nó là gì?

Hiệu ứng nổ emoji khi chọn ví từ danh sách ví 🌈🌈🌈. Nó sử dụng biểu tượng của ví bạn chọn.
