# Địa phương hoá

## Tùy chỉnh ngôn ngữ dApp của bạn với các bản dịch tích hợp sẵn

Mặc định 'CinaWalletKit' hỗ trợ locale `en-US` cho người dùng ngôn ngữ Tiếng Anh.

Nếu có, CinaWalletKit sẽ phát hiện [ngôn ngữ ưa thích](https://developer.mozilla.org/en-US/docs/Web/API/Navigator/language) của người dùng và chọn các bản dịch phù hợp. Các nhà phát triển luôn có thể thay đổi ngôn ngữ mặc định.

Để chỉ định ngôn ngữ cho người dùng của bạn, chỉ cần thêm `locale="zh-CN"` làm prop trong `CinaWalletKitProvider`

```tsx
import { CinaWalletKitProvider } from '@cinagroup/CinaWalletKit';

export const App = () => (
  <CinaWalletKitProvider locale="zh-CN" {...etc}>
    {/* Your App */}
  </CinaWalletKitProvider>
);
```

### Sử dụng với Next.js

Hỗ trợ địa phương hóa của 'CinaWalletKit' hoạt động tốt hơn với [Sub-path Routing](https://nextjs.org/docs/pages/building-your-application/routing/internationalization#sub-path-routing).

Cấu hình dự án Next.js của bạn như ví dụ dưới đây để thêm một sub-path `/locale/` tùy chọn vào các đường dẫn của bạn, điều này sẽ giúp các công cụ tìm kiếm và người dùng dễ dàng tìm kiếm hỗ trợ đa ngôn ngữ của bạn hơn.

```json
// next.config.js
{
  i18n: {
    locales: ['default', 'en', 'zh-CN'],
    defaultLocale: 'default',
  },
}
```

Sau đó, truyền `locale` được cung cấp bởi Pages Router vào `CinaWalletKitProvider`

```tsx
import { CinaWalletKitProvider, Locale } from '@cinagroup/CinaWalletKit';

export const App = () => {
  const { locale } = useRouter() as { locale: Locale };
  return (
    <CinaWalletKitProvider locale={locale} {...etc}>
      {/* Your App */}
    </CinaWalletKitProvider>
  )
};
```

Bạn có thể tham khảo một ví dụ cho Pages Router [tại đây](https://github.com/rainbow-me/CinaWalletKit/tree/main/examples/with-next).

App Router chưa hỗ trợ i18n. Tham khảo ví dụ riêng biệt của chúng tôi [tại đây](https://github.com/rainbow-me/CinaWalletKit/tree/main/examples/with-next-app-i18n) để có các thực tiễn tốt nhất về triển khai với middleware [`next-intl`](https://github.com/amannn/next-intl).

Chúng tôi khuyến nghị bạn sử dụng các kỹ thuật tương tự để dịch nội dung dApp của bạn cho việc hỗ trợ địa phương hóa đầy đủ. Các thư viện địa phương hóa như [`i18n-js`](https://github.com/fnando/i18n) và [`next-intl`](https://github.com/amannn/next-intl) cùng với các công cụ quản lý như Crowdin sẽ đơn giản hóa quá trình này.
