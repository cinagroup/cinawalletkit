# โหมดเท่ห์

## เปิดใช้งานโหมดเท่ห์ 🌈🌈🌈

คุณสามารถเปิดใช้งานโหมดเท่ห์ผ่านทาง prop `coolMode` บน `CinaWalletKitProvider`.

```tsx
import { CinaWalletKitProvider } from '@cinagroup/CinaWalletKit';

const App = () => (
  <CinaWalletKitProvider coolMode {...etc}>
    {/* Your App */}
  </CinaWalletKitProvider>
);
```

### แต่มันคืออะไร?

การระเบิดของอีโมจิเมื่อเลือกกระเป๋าเงินจากรายการกระเป๋าเงิน 🌈🌈🌈. ใช้สัญลักษณ์ของกระเป๋าเงินที่คุณเลือก.
