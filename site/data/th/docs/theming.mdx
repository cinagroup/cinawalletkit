# การตกแต่งธีม

## การใช้และปรับแต่งธีม

คุณสามารถปรับแต่ง CinaWalletKit UI ให้ตรงกับแบรนด์ของคุณได้ คุณสามารถเลือกจากสีเน้นและขนาดรัศมีขอบที่กำหนดล่วงหน้าได้เล็กน้อย

มีฟังก์ชันธีมในตัว 3 รูปแบบ:

- `lightTheme` &nbsp; (ค่าเริ่มต้น)
- `darkTheme`
- `midnightTheme`

ฟังก์ชันธีมจะคืนค่าเป็นวัตถุธีม คุณสามารถส่งวัตถุนี้ไปยัง `CinaWalletKitProvider` ตำแหน่ง `theme` ได้

```tsx
import { CinaWalletKitProvider, darkTheme } from '@cinagroup/CinaWalletKit';

export const App = () => (
  <CinaWalletKitProvider theme={darkTheme()} {...etc}>
    {/* Your App */}
  </CinaWalletKitProvider>
);
```

หากต้องการ คุณสามารถส่งค่า `accentColor`, `accentColorForeground`, `borderRadius`, `fontStack` และ `overlayBlur` เพื่อนำมาปรับแต่งได้

```tsx
import { CinaWalletKitProvider, darkTheme } from '@cinagroup/CinaWalletKit';

export const App = () => (
  <CinaWalletKitProvider
    theme={darkTheme({
      __accentColor__: '#7b3fe4',
      __accentColorForeground__: 'white',
      __borderRadius__: 'small',
      __fontStack__: 'system',
      __overlayBlur__: 'small',
    })}
    {...etc}
  >
    {/* Your App */}
  </CinaWalletKitProvider>
);
```

เช่น การปรับแต่งธีมเข้มด้วยสีเน้นสีม่วงและสเกลรัศมีขอบ `medium`:

```tsx
import { CinaWalletKitProvider, darkTheme } from '@cinagroup/CinaWalletKit';

const App = () => {
  return (
    <CinaWalletKitProvider
      theme={darkTheme({
        accentColor: '#7b3fe4',
        accentColorForeground: 'white',
        borderRadius: 'medium',
      })}
      {...etc}
    >
      {/* Your App */}
    </CinaWalletKitProvider>
  );
};
```

ในแต่ละธีมยังมีการตั้งค่าสีกำหนดไว้ล่วงหน้า (`blue`, `green`, `orange`, `pink`, `purple`, `red`) ที่สามารถนำไปใส่ในวัตถุออปชันได้ เช่น การใช้การตั้งค่าสีเน้น `pink`:

```tsx
import { CinaWalletKitProvider, darkTheme } from '@cinagroup/CinaWalletKit';

const App = () => {
  return (
    <CinaWalletKitProvider
      theme={darkTheme({
        ...darkTheme.accentColors.pink,
      })}
      {...etc}
    >
      {/* Your App */}
    </CinaWalletKitProvider>
  );
};
```

### ตัวอย่าง

#### ฟังก์ชันธีม

นี่คือวิธีต่างๆ ที่คุณสามารถใช้ `theme` prop ได้

ใช้ธีม `darkTheme`

```tsx
import { CinaWalletKitProvider, darkTheme } from '@cinagroup/CinaWalletKit';

export const App = () => (
  <CinaWalletKitProvider theme={darkTheme()} {...etc}>
    {/* Your App */}
  </CinaWalletKitProvider>
);
```

ใช้ธีม `midnightTheme`

```tsx
import {
  CinaWalletKitProvider,
  midnightTheme,
} from '@cinagroup/CinaWalletKit';

export const App = () => (
  <CinaWalletKitProvider theme={midnightTheme()} {...etc}>
    {/* Your App */}
  </CinaWalletKitProvider>
);
```

#### สีเน้น

นี่คือวิธีต่างๆ ที่คุณสามารถใช้การตั้งค่า `accentColor` ได้

ตั้งค่าสีเน้นเป็นค่าสีม่วงที่กำหนดเอง

```tsx
import { CinaWalletKitProvider, darkTheme } from '@cinagroup/CinaWalletKit';

const App = () => {
  return (
    <CinaWalletKitProvider
      theme={darkTheme({
        accentColor: '#7b3fe4',
        accentColorForeground: 'white',
      })}
      {...etc}
    >
      {/* Your App */}
    </CinaWalletKitProvider>
  );
};
```

ตั้งค่าสีเน้นเป็นการตั้งค่า `green` ที่กำหนดไว้ล่วงหน้า

```tsx
import { CinaWalletKitProvider, darkTheme } from '@cinagroup/CinaWalletKit';

const App = () => {
  return (
    <CinaWalletKitProvider
      theme={darkTheme({
        ...darkTheme.accentColors.green,
      })}
      {...etc}
    >
      {/* Your App */}
    </CinaWalletKitProvider>
  );
};
```

#### รัศมีขอบ

นี่คือวิธีต่างๆ ที่คุณสามารถใช้การตั้งค่า `borderRadius` ได้

ตั้งค่ารัศมีขอบเป็น `medium`

```tsx
import { CinaWalletKitProvider, darkTheme } from '@cinagroup/CinaWalletKit';

const App = () => {
  return (
    <CinaWalletKitProvider
      theme={darkTheme({
        borderRadius: 'medium',
      })}
      {...etc}
    >
      {/* Your App */}
    </CinaWalletKitProvider>
  );
};
```

ตั้งค่ารัศมีขอบเป็น `none`

```tsx
import { CinaWalletKitProvider, darkTheme } from '@cinagroup/CinaWalletKit';

const App = () => {
  return (
    <CinaWalletKitProvider
      theme={darkTheme({
        borderRadius: 'none',
      })}
      {...etc}
    >
      {/* Your App */}
    </CinaWalletKitProvider>
  );
};
```

> เตือนความจํา: ค่ารัศมีขอบที่สามารถใช้งานได้คือ: `large` (ค่าเริ่มต้น), `medium`, `small` และ `none`

#### สแต็กฟอนต์

โดยค่าเริ่มต้นนั้น `fontStack` จะตั้งค่าเป็น `rounded` นี่คือวิธีการใช้การตั้งค่า `fontStack`

ตั้งค่าสแต็กฟอนต์เป็น `system`

```tsx
import { CinaWalletKitProvider, darkTheme } from '@cinagroup/CinaWalletKit';

const App = () => {
  return (
    <CinaWalletKitProvider
      theme={darkTheme({
        fontStack: 'system',
      })}
      {...etc}
    >
      {/* Your App */}
    </CinaWalletKitProvider>
  );
};
```

#### การเบลอเลเยอร์ใน

โดยค่าเริ่มต้นนั้น `overlayBlur` จะตั้งค่าเป็น `none` นี่คือวิธีการใช้การตั้งค่า `overlayBlur`

ตั้งค่าเบลอเลเยอร์ในเป็น `small`

```tsx
import { CinaWalletKitProvider, darkTheme } from '@cinagroup/CinaWalletKit';

const App = () => {
  return (
    <CinaWalletKitProvider
      theme={darkTheme({
        overlayBlur: 'small',
      })}
      {...etc}
    >
      {/* Your App */}
    </CinaWalletKitProvider>
  );
};
```

#### ผสมและจับคู่

นี่คือวิธีต่างๆ ที่คุณสามารถใช้ธีมที่แตกต่างกันได้ พร้อมกับคุณสมบัติ `accentColor`, `borderRadius` และ `fontStack`

- ใช้ธีม `lightTheme`
- ตั้งค่าสีเน้นเป็นค่าสีม่วงที่กำหนดเอง
- ตั้งค่ารัศมีขอบเป็น `medium`
- ตั้งค่าสแต็กฟอนต์เป็น `system`

```tsx
import { CinaWalletKitProvider, lightTheme } from '@cinagroup/CinaWalletKit';

const App = () => {
  return (
    <CinaWalletKitProvider
      theme={lightTheme({
        accentColor: '#7b3fe4',
        accentColorForeground: 'white',
        borderRadius: 'medium',
        fontStack: 'system',
      })}
      {...etc}
    >
      {/* Your App */}
    </CinaWalletKitProvider>
  );
};
```

- ใช้ธีม `midnightTheme`
- ตั้งค่าสีเน้นเป็นการตั้งค่า `pink` ที่กำหนดไว้ล่วงหน้า
- ตั้งค่ารัศมีขอบเป็น `small`
- ตั้งค่าสแต็กฟอนต์เป็น `system`

```tsx
import {
  CinaWalletKitProvider,
  midnightTheme,
} from '@cinagroup/CinaWalletKit';

const App = () => {
  return (
    <CinaWalletKitProvider
      theme={midnightTheme({
        ...midnightTheme.accentColors.pink,
        borderRadius: 'small',
        fontStack: 'system',
      })}
      {...etc}
    >
      {/* Your App */}
    </CinaWalletKitProvider>
  );
};
```

#### รองรับโหมดมืด

หากแอปของคุณใช้คิวรี่สื่อ `prefers-color-scheme: dark` มาตรฐานเพื่อสลับระหว่างโหมดสว่างและโหมดเข้ม คุณสามารถกำหนดวัตถุธีมไดนามิกที่มีค่า `lightMode` และ `darkMode` ได้ตามต้องการ

```tsx
import {
  CinaWalletKitProvider,
  lightTheme,
  darkTheme,
} from '@cinagroup/CinaWalletKit';

const App = () => (
  <CinaWalletKitProvider
    theme={{
      lightMode: lightTheme(),
      darkMode: darkTheme(),
    }}
    {...etc}
  >
    {/* Your App */}
  </CinaWalletKitProvider>
);
```
