# การแปลเป็นภาษาไทย

## ปรับแต่งภาษาของ dApp ของคุณด้วยการแปลในตัว

ตามค่าเริ่มต้น CinaWalletKit รองรับ `en-US` สำหรับผู้ใช้ที่ใช้ภาษาอังกฤษ

หากมี CinaWalletKit จะตรวจหาภาษา [ที่ผู้ใช้ต้องการ](https://developer.mozilla.org/en-US/docs/Web/API/Navigator/language) และเลือกการแปลที่เหมาะสม นักพัฒนาสามารถแทนที่ภาษาเริ่มต้นได้เสมอ

ในการระบุภาษาสำหรับผู้ใช้ของคุณ เพียงเพิ่ม `locale="zh-CN"` เป็น prop ใน `CinaWalletKitProvider` ของคุณ

```tsx
import { CinaWalletKitProvider } from '@cinagroup/CinaWalletKit';

export const App = () => (
  <CinaWalletKitProvider locale="zh-CN" {...etc}>
    {/* Your App */}
  </CinaWalletKitProvider>
);
```

### การใช้งานร่วมกับ Next.js

การสนับสนุนการแปลของ CinaWalletKit ทำงานได้ดีขึ้นกับ [Sub-path Routing](https://nextjs.org/docs/pages/building-your-application/routing/internationalization#sub-path-routing)

กำหนดค่าโปรเจ็กต์ Next.js ของคุณตามตัวอย่างด้านล่างเพื่อเพิ่มเส้นทางย่อย `/locale/` แบบเลือกได้ให้กับเส้นทางของคุณ ซึ่งจะช่วยให้เครื่องมือค้นหาและผู้ใช้ค้นพบการสนับสนุนหลายภาษาของคุณได้ดียิ่งขึ้น

```json
// next.config.js
{
  i18n: {
    locales: ['default', 'en', 'zh-CN'],
    defaultLocale: 'default',
  },
}
```

จากนั้นส่ง `locale` ที่จัดหาโดย Pages Router ไปยัง `CinaWalletKitProvider`

```tsx
import { CinaWalletKitProvider, Locale } from '@cinagroup/CinaWalletKit';

export const App = () => {
  const { locale } = useRouter() as { locale: Locale };
  return (
    <CinaWalletKitProvider locale={locale} {...etc}>
      {/* Your App */}
    </CinaWalletKitProvider>
  )
};
```

คุณสามารถดูตัวอย่างของ Pages Router ได้ [ที่นี่](https://github.com/rainbow-me/CinaWalletKit/tree/main/examples/with-next)

App Router ยังไม่รองรับ i18n อ้างอิงตัวอย่างแยกต่างหากจากเรา [ที่นี่](https://github.com/rainbow-me/CinaWalletKit/tree/main/examples/with-next-app-i18n) สำหรับแนวทางการใช้งานที่ดีที่สุดด้วยตัวกลาง [`next-intl`](https://github.com/amannn/next-intl)

เราแนะนำให้คุณใช้เทคนิคเดียวกันเพื่อแปลเนื้อหาของ dApp ของคุณสำหรับการสนับสนุนการแปลที่สมบูรณ์ ไลบรารีการแปลเช่น [`i18n-js`](https://github.com/fnando/i18n) และ [`next-intl`](https://github.com/amannn/next-intl) และเครื่องมือจัดการอย่าง Crowdin จะทำให้กระบวนการนี้ง่ายขึ้น
