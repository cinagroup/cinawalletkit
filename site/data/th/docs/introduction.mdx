# CinaWalletKit

## วิธีที่ดีที่สุดในการเชื่อมต่อกระเป๋าเงิน 🌈

<Video src="https://res.cloudinary.com/rainbowme/video/upload/s--aTmEjSrr--/f_auto:video,q_auto/v1/CinaWalletKit/site-assets/CinaWalletKit-video-cutdown-edit_gy0q16" />

CinaWalletKit เป็นไลบรารี [React](https://reactjs.org/) ที่ทำให้การเพิ่มการเชื่อมต่อกระเป๋าสตางค์เข้ากับ dapp ของคุณเป็นเรื่องง่าย มีการทำงานที่เข้าใจง่าย, ตอบสนอง และสามารถปรับแต่งได้ตามใจชอบ

### คุณสมบัติ

#### การจัดการกระเป๋าสตางค์

การจัดการกระเป๋าสตางค์ให้กับ dapp ของคุณจากกล่องนอก นอกจากเชื่อมต่อและตัดการเชื่อมต่อของกระเป๋าสตางค์แล้ว CinaWalletKit ยังสนับสนุนกระเป๋าที่หลากหลาย, การเปลี่ยนสายการเชื่อมต่อ, การแก้ไขที่อยู่ให้เป็น ENS, แสดงรายการยอดเงินคงเหลือ และอีกมากมาย!

#### ปรับแต่งได้

คุณสามารถปรับแต่ง UI ของ CinaWalletKit เพื่อให้เข้ากับแบรนด์ของคุณ คุณสามารถเลือกจากสีเน้นที่กำหนดไว้ล่วงหน้า และการตั้งค่าขอบโค้ง สำหรับการใช้งานที่ซับซ้อนขึ้น คุณสามารถกำหนดธีมแบบปรับได้เอง, แสดงปุ่มของคุณเอง และละเว้นคุณสมบัติบางอย่าง การใช้งานโหมดมืดสามารถรวมได้

#### มาตรฐานอุตสาหกรรม

เพื่อการผสานการทำงานที่ดียิ่งขึ้นกับผลิตภัณฑ์ส่วนใหญ่ เราพึ่งพา [viem](https://viem.sh) และ [wagmi](https://wagmi.sh/) — ไลบรารีที่ใช้กันทั่วไปที่สุดในพื้นที่นี้

มาตรฐานกระเป๋าสตางค์เช่น EIP-1193 และ EIP-6963 มีการสนับสนุนจากกล่องนอกทำให้ผู้ใช้งานกระเป๋าบราวเซอร์เชื่อมต่อกระเป๋าของพวกเขากับ dApp ของคุณได้อย่างไม่ยาก

### ชุมชน

เรารู้สึกตื่นเต้นที่จะได้เห็นชุมชนรับรู้ CinaWalletKit, รายงานปัญหา และให้ข้อเสนอแนะ ไม่ว่าจะเป็นการร้องขอฟีเจอร์, รายงานบั๊ก หรือโปรเจ็กต์ที่ต้องการนำเสนอก็สามารถเข้าร่วมได้!
