# Mode Cool

## Activation du mode cool 🌈🌈🌈

Vous pouvez activer le mode cool via la propriété `coolMode` sur le `CinaWalletKitProvider`.

```tsx
import { CinaWalletKitProvider } from '@cinagroup/CinaWalletKit';

const App = () => (
  <CinaWalletKitProvider coolMode {...etc}>
    {/* Your App */}
  </CinaWalletKitProvider>
);
```

### Mais qu'est-ce que c'est ?

Explosions d'émoticônes lorsqu'un portefeuille est sélectionné dans la liste des portefeuilles 🌈🌈🌈. Il utilise l'icône du portefeuille que vous sélectionnez.
