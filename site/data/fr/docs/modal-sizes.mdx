# Tailles de la fenêtre modale

## Taille Large vs Compact

Par défaut, la `modalSize` de CinaWalletKit est réglée sur `large` (et ressemble à ceci) :

<Img src="/modal-default.png" height="569" width="1029" />

Nous proposons également une taille de fenêtre modale `compact`, qui ressemble à ceci :

<Img src="/modal-compact.png" height="569" width="1029" />

Pour définir la taille de votre fenêtre modale sur `compact`, ajoutez simplement `modalSize="compact"` comme prop dans votre `CinaWalletKitProvider` :

```tsx
import { CinaWalletKitProvider } from '@cinagroup/CinaWalletKit';

export const App = () => (
  <CinaWalletKitProvider modalSize="compact" {...etc}>
    {/* Your App */}
  </CinaWalletKitProvider>
);
```
