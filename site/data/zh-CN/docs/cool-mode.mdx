# 酷模式

## 启用酷模式 🌈🌈🌈

您可以通过 `CinaWalletKitProvider` 上的 `coolMode` 属性启用酷模式。

```tsx
import { CinaWalletKitProvider } from '@cinagroup/CinaWalletKit';

const App = () => (
  <CinaWalletKitProvider coolMode {...etc}>
    {/* Your App */}
  </CinaWalletKitProvider>
);
```

### 但这是什么？

选择钱包列表中的某个钱包时会产生表情符号爆炸 🌈🌈🌈。 它使用您选择的钱包的图标。
