# Modo Legal

## Habilitando o modo legal 🌈🌈🌈

Você pode habilitar o modo legal via a prop `coolMode` no `CinaWalletKitProvider`.

```tsx
import { CinaWalletKitProvider } from '@cinagroup/CinaWalletKit';

const App = () => (
  <CinaWalletKitProvider coolMode {...etc}>
    {/* Your App */}
  </CinaWalletKitProvider>
);
```

### Mas o que é isso?

Explosões de emoji quando uma carteira é selecionada da lista de carteiras 🌈🌈🌈. Usa o ícone da carteira que você selecionar.
