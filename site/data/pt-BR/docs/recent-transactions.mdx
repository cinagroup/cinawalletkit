# Transações Recentes

## Mostre suas transações recentes

Você pode optar por exibir transações recentes no modal de conta do CinaWalletKit. Note que todas as transações são mantidas no armazenamento local e devem ser registradas manualmente no CinaWalletKit para serem exibidas.

A implementação padrão do `ConnectButton` também exibirá um indicador de carregamento ao redor do avatar do usuário se houver transações pendentes. Implementações personalizadas do `ConnectButton` podem recriar esse comportamento usando a propriedade `account.hasPendingTransactions` que é passada para sua função de renderização.

Para utilizar este recurso, primeiro ative a opção `showRecentTransactions` no `CinaWalletKitProvider`.

```tsx
import { CinaWalletKitProvider } from '@cinagroup/CinaWalletKit';

const App = () => {
  return (
    <CinaWalletKitProvider showRecentTransactions={true} {...etc}>
      {/* ... */}
    </CinaWalletKitProvider>
  );
};
```

As transações podem ser registradas no CinaWalletKit usando o gancho `useAddRecentTransaction`.

```tsx
import { useAddRecentTransaction } from '@cinagroup/CinaWalletKit';

export default () => {
  const addRecentTransaction = useAddRecentTransaction();

  return (
    <button
      onClick={() => {
        addRecentTransaction({
          hash: '0x...',
          description: '...',
        });
      }}
    >
      Add recent transaction
    </button>
  );
};
```

Uma vez que uma transação foi registrada no CinaWalletKit, seu status será atualizado após a conclusão.

Por padrão, a transação será considerada concluída assim que um único bloco for minerado sobre o bloco em que a transação foi minerada, mas isso pode ser configurado especificando um valor personalizado de `confirmations`.

```tsx
import { useAddRecentTransaction } from '@cinagroup/CinaWalletKit';

export default () => {
  const addRecentTransaction = useAddRecentTransaction();

  return (
    <button
      onClick={() => {
        addRecentTransaction({
          hash: '0x...',
          description: '...',
          confirmations: 100,
        });
      }}
    >
      Add recent transaction
    </button>
  );
};
```
