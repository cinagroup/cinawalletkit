# Informações Personalizadas do Aplicativo

## Personalizando as informações do seu aplicativo

Você pode passar as informações do seu aplicativo na propriedade `appInfo` do `CinaWalletKitProvider`. As propriedades que você pode modificar são o nome do seu aplicativo (`appName`) e o link para onde o botão "Saiba Mais" no modal de conexão redireciona (`learnMoreUrl`):

```tsx
import { CinaWalletKitProvider } from '@cinagroup/CinaWalletKit';

const App = () => {
  return (
    <CinaWalletKitProvider
      appInfo={{
        appName: 'CinaWalletKit Demo',
        learnMoreUrl: 'https://learnaboutcryptowallets.example',
      }}
      {...etc}
    >
      {/* ... */}
    </CinaWalletKitProvider>
  );
};
```

### Adicionar um Aviso Legal

Você pode fornecer seu próprio aviso legal para ser exibido na parte inferior do modal de conexão. A propriedade `disclaimer` em `appInfo` aceita um `DisclaimerComponent` com argumentos `Text` e `Link`. Você pode usá-lo para construir seu próprio aviso legal que herda o sistema de temas. Nós exportamos o tipo `DisclaimerComponent` para usuários do TypeScript:

```tsx
import {
  CinaWalletKitProvider,
  DisclaimerComponent,
} from '@cinagroup/CinaWalletKit';

const Disclaimer: DisclaimerComponent = ({ Text, Link }) => (
  <Text>
    By connecting your wallet, you agree to the{' '}
    <Link href="https://termsofservice.xyz">Terms of Service</Link> and
    acknowledge you have read and understand the protocol{' '}
    <Link href="https://disclaimer.xyz">Disclaimer</Link>
  </Text>
);

const App = () => {
  return (
    <CinaWalletKitProvider
      appInfo={{
        appName: 'CinaWalletKit Demo',
        disclaimer: Disclaimer,
      }}
      {...etc}
    >
      {/* ... */}
    </CinaWalletKitProvider>
  );
};
```
