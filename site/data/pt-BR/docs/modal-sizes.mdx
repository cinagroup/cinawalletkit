# Tamanhos de Modal

## Tamanho Largo vs. Compacto

Por padrão, o `modalSize` do CinaWalletKit é configurado como `wide` (e se parece com isto):

<Img src="/modal-default.png" height="569" width="1029" />

Também fornecemos um tamanho de modal `compacto`, que se parece com isto:

<Img src="/modal-compact.png" height="569" width="1029" />

Para definir seu tamanho de modal para `compacto`, basta adicionar `modalSize="compact"` como uma propriedade em seu `CinaWalletKitProvider`:

```tsx
import { CinaWalletKitProvider } from '@cinagroup/CinaWalletKit';

export const App = () => (
  <CinaWalletKitProvider modalSize="compact" {...etc}>
    {/* Your App */}
  </CinaWalletKitProvider>
);
```
