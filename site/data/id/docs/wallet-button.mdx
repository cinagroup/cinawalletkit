# TombolWallet

> Catatan: `TombolWallet` saat ini mengandalkan standar dompet EIP-1193, tetapi akan segera mendukung EIP-6963.

Komponen baru `TombolWallet` membantu dApps dengan implementasi daftar dompet kustom mengadopsi CinaWalletKit dan semua keuntungannya yang bebas perawatan.

```tsx
import { WalletButton } from '@cinagroup/CinaWalletKit';

<WalletButton wallet="rainbow" />
<WalletButton wallet="metamask" />
<WalletButton wallet="base" />
// Backwards-compatible wallet ids: "baseAccount" and "coinbaseWallet"
```

Seperti `TombolConnect`, komponen `TombolWallet.Custom` tersedia untuk implementasi dan gaya kustom.

```tsx
<WalletButton.Custom wallet="rainbow">
  {({ ready, connect }) => {
    return (
      <button
        type="button"
        disabled={!ready}
        onClick={connect}
      >
        Connect Rainbow
      </button>
    );
  }}
</WalletButton.Custom>
```

Sebagian besar dApps terbaik dilayani oleh [TombolConnect](https://www.CinaWalletKit.com/docs/connect-button). Rujuk dokumen [di sini](https://www.CinaWalletKit.com/docs/wallet-button) untuk informasi lebih lanjut tentang adopsi dan kasus penggunaan `TombolWallet`.
