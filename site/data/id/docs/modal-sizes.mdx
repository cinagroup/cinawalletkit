# Ukuran Modal

## Ukuran luas vs. kompak

Secara default, ukuran `modalSize` CinaWalletKit diatur ke `luas` (dan tampil seperti ini):

<Img src="/modal-default.png" height="569" width="1029" />

Kami juga menyediakan ukuran modal `kompak`, yang tampil seperti ini:

<Img src="/modal-compact.png" height="569" width="1029" />

Untuk mengatur ukuran modal ke `kompak`, cukup tambahkan `modalSize="kompak"` sebagai prop dalam `CinaWalletKitProvider` Anda:

```tsx
import { CinaWalletKitProvider } from '@cinagroup/CinaWalletKit';

export const App = () => (
  <CinaWalletKitProvider modalSize="compact" {...etc}>
    {/* Your App */}
  </CinaWalletKitProvider>
);
```
