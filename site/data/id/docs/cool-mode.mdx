# Mode Keren

## Mengaktifkan mode keren 🌈🌈🌈

Anda dapat mengaktifkan mode keren melalui prop `coolMode` pada `CinaWalletKitProvider`.

```tsx
import { CinaWalletKitProvider } from '@cinagroup/CinaWalletKit';

const App = () => (
  <CinaWalletKitProvider coolMode {...etc}>
    {/* Your App */}
  </CinaWalletKitProvider>
);
```

### Tetapi apa itu?

Ledakan emoji saat dompet dipilih dari daftar dompet 🌈🌈🌈. Ini menggunakan ikon dompet yang Anda pilih.
