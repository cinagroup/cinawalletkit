# Info Aplikasi Kustom

## Menyesuaikan info aplikasi Anda

Anda dapat memasukkan info aplikasi Anda dalam properti `appInfo` untuk `CinaWalletKitProvider`. Properti yang dapat Anda modifikasi termasuk nama aplikasi Anda (`appName`) dan tautan di mana tombol “Pelajari Lebih Lanjut” di modal koneksi akan diarahkan (`learnMoreUrl`):

```tsx
import { CinaWalletKitProvider } from '@cinagroup/CinaWalletKit';

const App = () => {
  return (
    <CinaWalletKitProvider
      appInfo={{
        appName: 'CinaWalletKit Demo',
        learnMoreUrl: 'https://learnaboutcryptowallets.example',
      }}
      {...etc}
    >
      {/* ... */}
    </CinaWalletKitProvider>
  );
};
```

### Tambahkan Disclaimer

Anda dapat menyediakan disclaimer Anda sendiri yang akan ditampilkan di bagian bawah modal koneksi. Properti `disclaimer` dalam `appInfo` menerima sebuah `DisclaimerComponent` dengan argumen `Text` dan `Link`. Anda dapat menggunakannya untuk membuat disclaimer sendiri yang mewarisi sistem tema. Kami mengekspor tipe `DisclaimerComponent` untuk pengguna TypeScript:

```tsx
import {
  CinaWalletKitProvider,
  DisclaimerComponent,
} from '@cinagroup/CinaWalletKit';

const Disclaimer: DisclaimerComponent = ({ Text, Link }) => (
  <Text>
    By connecting your wallet, you agree to the{' '}
    <Link href="https://termsofservice.xyz">Terms of Service</Link> and
    acknowledge you have read and understand the protocol{' '}
    <Link href="https://disclaimer.xyz">Disclaimer</Link>
  </Text>
);

const App = () => {
  return (
    <CinaWalletKitProvider
      appInfo={{
        appName: 'CinaWalletKit Demo',
        disclaimer: Disclaimer,
      }}
      {...etc}
    >
      {/* ... */}
    </CinaWalletKitProvider>
  );
};
```
