# Son İşlemler

## Son işlemlerinizi göster

CinaWalletKit'in hesap modali içinde son işlemleri görüntüleme opsiyonuna katılabilirsiniz. Tüm işlemlerin yerel depolamada tutulduğunu ve görüntülenmesi için CinaWalletKit ile manuel olarak kaydedilmesi gerektiğini unutmayın.

Varsayılan `ConnectButton` uygulaması, bekleyen işlemler varsa, kullanıcının avatarının etrafında bir yükleme göstergesi de gösterecektir. Özel `ConnectButton` uygulamaları, render fonksiyonunuza aktarılan `account.hasPendingTransactions` özelliği aracılığıyla bu davranışı yeniden oluşturabilir.

Bu özelliği kullanmak için, önce `CinaWalletKitProvider` üzerinde `showRecentTransactions` seçeneğini etkinleştirin.

```tsx
import { CinaWalletKitProvider } from '@cinagroup/CinaWalletKit';

const App = () => {
  return (
    <CinaWalletKitProvider showRecentTransactions={true} {...etc}>
      {/* ... */}
    </CinaWalletKitProvider>
  );
};
```

İşlemler daha sonra `useAddRecentTransaction` kancası kullanılarak CinaWalletKit ile kaydedilebilir.

```tsx
import { useAddRecentTransaction } from '@cinagroup/CinaWalletKit';

export default () => {
  const addRecentTransaction = useAddRecentTransaction();

  return (
    <button
      onClick={() => {
        addRecentTransaction({
          hash: '0x...',
          description: '...',
        });
      }}
    >
      Add recent transaction
    </button>
  );
};
```

Bir işlem CinaWalletKit ile kaydedildikten sonra, tamamlandığında durumu güncellenecektir.

Varsayılan olarak, işlem, işlemin işlendiği bloğun üzerine bir tek blok madenciliği yapıldığında tamamlanmış kabul edilir, ancak bu, özel bir `confirmations` değeri belirleyerek yapılandırılabilir.

```tsx
import { useAddRecentTransaction } from '@cinagroup/CinaWalletKit';

export default () => {
  const addRecentTransaction = useAddRecentTransaction();

  return (
    <button
      onClick={() => {
        addRecentTransaction({
          hash: '0x...',
          description: '...',
          confirmations: 100,
        });
      }}
    >
      Add recent transaction
    </button>
  );
};
```
