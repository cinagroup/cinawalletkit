# Havalı Mod

## Havalı modu etkinleştirme 🌈🌈🌈

Havalı modu `CinaWalletKitProvider`'da `coolMode` özelliği ile etkinleştirebilirsiniz.

```tsx
import { CinaWalletKitProvider } from '@cinagroup/CinaWalletKit';

const App = () => (
  <CinaWalletKitProvider coolMode {...etc}>
    {/* Your App */}
  </CinaWalletKitProvider>
);
```

### Ama nedir bu?

Cüzdan listesinden bir cüzdan seçildiğinde emoji patlamaları 🌈🌈🌈. Seçtiğiniz cüzdanın simgesini kullanır.
