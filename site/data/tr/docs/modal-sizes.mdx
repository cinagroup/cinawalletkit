# Modal Boyutları

## Geniş vs. Kompakt boyut

Varsayılan olarak, CinaWalletKit'in `modalSize` değeri `wide` olarak ayarlanmıştır (aşağıdaki gibi görünür):

<Img src="/modal-default.png" height="569" width="1029" />

Ayrıca, aşağıdakine benzeyen bir `compact` modal boyutu da sağlıyoruz:

<Img src="/modal-compact.png" height="569" width="1029" />

Modunuzun boyutunu `compact` olarak ayarlamak için `CinaWalletKitProvider`'ınıza `modalSize="compact"` parametresini ekleyin:

```tsx
import { CinaWalletKitProvider } from '@cinagroup/CinaWalletKit';

export const App = () => (
  <CinaWalletKitProvider modalSize="compact" {...etc}>
    {/* Your App */}
  </CinaWalletKitProvider>
);
```
