# Özel Uygulama Bilgileri

## Uygulamanızın bilgilerini özelleştirme

Uygulamanızın bilgilerini `CinaWalletKitProvider` üzerindeki `appInfo` özelliği ile iletebilirsiniz. Özelleştirebileceğiniz özellikler, uygulamanızın adı (`appName`) ve bağlantı modülündeki 'Daha Fazla Bilgi’ düğmesinin yönlendirdiği bağlantı (`learnMoreUrl`) dır:

```tsx
import { CinaWalletKitProvider } from '@cinagroup/CinaWalletKit';

const App = () => {
  return (
    <CinaWalletKitProvider
      appInfo={{
        appName: 'CinaWalletKit Demo',
        learnMoreUrl: 'https://learnaboutcryptowallets.example',
      }}
      {...etc}
    >
      {/* ... */}
    </CinaWalletKitProvider>
  );
};
```

### Bir Feragatname Ekleyin

Bağlantı modülünün en altında gösterilecek kendi feragatnamenizi sağlayabilirsiniz. `appInfo`'daki `disclaimer` özelliği, `Text` ve `Link` argümanlarıyla bir `DisclaimerComponent` alır. Kendi temanızı miras alan bir feragatname oluşturmak için bunu kullanabilirsiniz. TypeScript kullanıcıları için `DisclaimerComponent` türünü dışa aktarıyoruz:

```tsx
import {
  CinaWalletKitProvider,
  DisclaimerComponent,
} from '@cinagroup/CinaWalletKit';

const Disclaimer: DisclaimerComponent = ({ Text, Link }) => (
  <Text>
    By connecting your wallet, you agree to the{' '}
    <Link href="https://termsofservice.xyz">Terms of Service</Link> and
    acknowledge you have read and understand the protocol{' '}
    <Link href="https://disclaimer.xyz">Disclaimer</Link>
  </Text>
);

const App = () => {
  return (
    <CinaWalletKitProvider
      appInfo={{
        appName: 'CinaWalletKit Demo',
        disclaimer: Disclaimer,
      }}
      {...etc}
    >
      {/* ... */}
    </CinaWalletKitProvider>
  );
};
```
