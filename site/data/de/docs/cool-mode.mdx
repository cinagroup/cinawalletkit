# Coole Modus

## Aktivierung des coolen Modus 🌈🌈🌈

Sie können den coolen Modus über das `coolMode`-Prop im `CinaWalletKitProvider` aktivieren.

```tsx
import { CinaWalletKitProvider } from '@cinagroup/CinaWalletKit';

const App = () => (
  <CinaWalletKitProvider coolMode {...etc}>
    {/* Your App */}
  </CinaWalletKitProvider>
);
```

### Aber was ist das?

Emoji-Explosionen, wenn ein Wallet aus der Wallet-Liste ausgewählt wird 🌈🌈🌈. Es verwendet das Symbol des ausgewählten Wallets.
