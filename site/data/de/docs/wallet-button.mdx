# WalletButton

> Hinweis: Der `WalletButton` basiert derzeit auf dem EIP-1193-Wallet-Standard, wird jedoch bald EIP-6963 unterstützen.

Der neue `WalletButton`-Komponente hilft dApps mit benutzerdefinierten Wallet-Listen-Implementierungen, CinaWalletKit und all seine wartungsfreien Vorteile zu übernehmen.

```tsx
import { WalletButton } from '@cinagroup/CinaWalletKit';

<WalletButton wallet="rainbow" />
<WalletButton wallet="metamask" />
<WalletButton wallet="base" />
// Backwards-compatible wallet ids: "baseAccount" and "coinbaseWallet"
```

Wie der `ConnectButton` ist die `WalletButton.Custom`-Komponente für benutzerdefinierte Implementierungen und Stilanpassungen verfügbar.

```tsx
<WalletButton.Custom wallet="rainbow">
  {({ ready, connect }) => {
    return (
      <button
        type="button"
        disabled={!ready}
        onClick={connect}
      >
        Connect Rainbow
      </button>
    );
  }}
</WalletButton.Custom>
```

Die meisten dApps profitieren am meisten vom [ConnectButton](https://www.CinaWalletKit.com/docs/connect-button). Weitere Informationen zur Einführung und den Anwendungsfällen des `WalletButton` finden Sie [hier](https://www.CinaWalletKit.com/docs/wallet-button).
