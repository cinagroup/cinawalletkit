# Benutzerdefinierte App-Informationen

## Anpassen der Informationen Ihrer App

Sie können die Informationen Ihrer App in der `appInfo` Eigenschaft von `CinaWalletKitProvider` angeben. Eigenschaften, die Sie ändern können, sind der Name Ihrer App (`appName`) und der Link, zu dem der „Mehr erfahren“ Button im Verbindungsmodul weiterleitet (`learnMoreUrl`):

```tsx
import { CinaWalletKitProvider } from '@cinagroup/CinaWalletKit';

const App = () => {
  return (
    <CinaWalletKitProvider
      appInfo={{
        appName: 'CinaWalletKit Demo',
        learnMoreUrl: 'https://learnaboutcryptowallets.example',
      }}
      {...etc}
    >
      {/* ... */}
    </CinaWalletKitProvider>
  );
};
```

### Haftungsausschluss hinzufügen

Sie können einen eigenen Haftungsausschluss bereitstellen, der unten im Verbindungsmodul angezeigt wird. Die `disclaimer` Eigenschaft in `appInfo` nimmt eine `DisclaimerComponent` mit den Argumenten `Text` und `Link`. Sie können es verwenden, um Ihren eigenen Haftungsausschluss zu erstellen, der das Themensystem erbt. Wir exportieren den `DisclaimerComponent` Typ für TypeScript Nutzer:

```tsx
import {
  CinaWalletKitProvider,
  DisclaimerComponent,
} from '@cinagroup/CinaWalletKit';

const Disclaimer: DisclaimerComponent = ({ Text, Link }) => (
  <Text>
    By connecting your wallet, you agree to the{' '}
    <Link href="https://termsofservice.xyz">Terms of Service</Link> and
    acknowledge you have read and understand the protocol{' '}
    <Link href="https://disclaimer.xyz">Disclaimer</Link>
  </Text>
);

const App = () => {
  return (
    <CinaWalletKitProvider
      appInfo={{
        appName: 'CinaWalletKit Demo',
        disclaimer: Disclaimer,
      }}
      {...etc}
    >
      {/* ... */}
    </CinaWalletKitProvider>
  );
};
```
