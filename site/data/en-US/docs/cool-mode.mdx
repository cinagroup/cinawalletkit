# Cool Mode

## Enabling cool mode 🌈🌈🌈

You can enable cool mode via the `coolMode` prop on the `CinaWalletKitProvider`.

```tsx
import { CinaWalletKitProvider } from '@cinagroup/CinaWalletKit';

const App = () => (
  <CinaWalletKitProvider coolMode {...etc}>
    {/* Your App */}
  </CinaWalletKitProvider>
);
```

### But what is it?

Emoji explosions when a wallet is selected from the wallet list 🌈🌈🌈. It uses the icon of the wallet you select.
