# कूल मोड

## कूल मोड सक्षम करना 🌈🌈🌈

आप `coolMode` प्रॉप का उपयोग करके `CinaWalletKitProvider` पर कूल मोड सक्षम कर सकते हैं।

```tsx
import { CinaWalletKitProvider } from '@cinagroup/CinaWalletKit';

const App = () => (
  <CinaWalletKitProvider coolMode {...etc}>
    {/* Your App */}
  </CinaWalletKitProvider>
);
```

### लेकिन यह क्या है?

इमोजी विस्फोट तब होते हैं जब वॉलेट सूची से एक वॉलेट चुना जाता है 🌈🌈🌈। यह आपके चयनित वॉलेट के आइकन का उपयोग करता है।
