# Tamaños de Modal

## Tamaño Ancho vs. Compacto

Por defecto, el `modalSize` de CinaWalletKit está configurado como `wide` (y se ve así):

<Img src="/modal-default.png" height="569" width="1029" />

También ofrecemos un tamaño de modal `compact`, que se ve así:

<Img src="/modal-compact.png" height="569" width="1029" />

Para configurar el tamaño de tu modal a `compact`, solo añade `modalSize="compact"` como un prop en tu `CinaWalletKitProvider`:

```tsx
import { CinaWalletKitProvider } from '@cinagroup/CinaWalletKit';

export const App = () => (
  <CinaWalletKitProvider modalSize="compact" {...etc}>
    {/* Your App */}
  </CinaWalletKitProvider>
);
```
