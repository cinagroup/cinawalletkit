# Modo Cool

## Activar modo cool 🌈🌈🌈

Puedes activar el modo cool a través de la propiedad `coolMode` en el `CinaWalletKitProvider`.

```tsx
import { CinaWalletKitProvider } from '@cinagroup/CinaWalletKit';

const App = () => (
  <CinaWalletKitProvider coolMode {...etc}>
    {/* Your App */}
  </CinaWalletKitProvider>
);
```

### ¿Pero qué es?

Explosiones de emojis cuando se selecciona una billetera de la lista de billeteras 🌈🌈🌈. Utiliza el icono de la billetera que selecciones.
