# CinaWalletKit

## Найкращий спосіб підключення гаманця 🌈

<Video src="https://res.cloudinary.com/rainbowme/video/upload/s--aTmEjSrr--/f_auto:video,q_auto/v1/CinaWalletKit/site-assets/CinaWalletKit-video-cutdown-edit_gy0q16" />

CinaWalletKit — це бібліотека [React](https://reactjs.org/), яка полегшує підключення гаманця до вашого децентралізованого додатку. Вона інтуїтивно зрозуміла, адаптивна та налаштовувана.

### Функції

#### Управління гаманцем

Автономне управління гаманцем для вашого децентралізованого додатку. Окрім обробки підключення та відключення гаманців, CinaWalletKit підтримує численні гаманці, змінює мережі підключення, перетворює адреси на ENS, відображає баланс і багато іншого!

#### Настроюваність

Ви можете налаштувати інтерфейс CinaWalletKit для відповідності вашому бренду. Ви можете вибрати з декількох попередньо визначених акцентних кольорів та конфігурацій радіуса меж. Для більш просунутих випадків використання, ви можете надати повністю кастомну тему, відобразити власну кнопку та опустити певні функції. Темний режим включено.

#### Промислові стандарти

Для кращої сумісності з більшістю продуктів ми покладаємось на [viem](https://viem.sh) та [wagmi](https://wagmi.sh/) — найпоширеніші бібліотеки у цій галузі.

Стандарти гаманців, такі як EIP-1193 та EIP-6963, підтримуються "зі старту", що робить підключення гаманця до вашого децентралізованого додатку ще простішим для користувачів браузерних гаманців.
