# Режим Cool

## Увімкнення режиму Cool 🌈🌈🌈

Ви можете увімкнути режим Cool через властивість `coolMode` в `CinaWalletKitProvider`.

```tsx
import { CinaWalletKitProvider } from '@cinagroup/CinaWalletKit';

const App = () => (
  <CinaWalletKitProvider coolMode {...etc}>
    {/* Your App */}
  </CinaWalletKitProvider>
);
```

### Але що це таке?

Експлозії емодзі, коли вибирається гаманець зі списку гаманців 🌈🌈🌈. Використовується іконка обраного вами гаманця.

#### Демо

Натисніть кнопки нижче, щоб побачити, що робить режим Cool.
