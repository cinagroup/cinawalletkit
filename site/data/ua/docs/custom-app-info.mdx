# Інформація про спеціальний додаток

## Налаштування інформації про ваш додаток

Ви можете передати інформацію про ваш додаток у властивості `appInfo` для `CinaWalletKitProvider`. Властивості, які можна змінити - це назва вашого додатка (`appName`) і посилання, куди перенаправляє кнопка "Дізнатися більше" в модулі підключення (`learnMoreUrl`):

```tsx
import { CinaWalletKitProvider } from '@cinagroup/CinaWalletKit';

const App = () => {
  return (
    <CinaWalletKitProvider
      appInfo={{
        appName: 'CinaWalletKit Demo',
        learnMoreUrl: 'https://learnaboutcryptowallets.example',
      }}
      {...etc}
    >
      {/* ... */}
    </CinaWalletKitProvider>
  );
};
```

### Додайте застереження

Ви можете надати власне застереження, яке буде відображатися внизу модального вікна з'єднання. Властивість `disclaimer` у `appInfo` приймає `DisclaimerComponent` з аргументами `Text` та `Link`. Ви можете використовувати його, щоб створити власне застереження, що успадковує систему темізації. Ми експортуємо тип `DisclaimerComponent` для користувачів TypeScript:

```tsx
import {
  CinaWalletKitProvider,
  DisclaimerComponent,
} from '@cinagroup/CinaWalletKit';

const Disclaimer: DisclaimerComponent = ({ Text, Link }) => (
  <Text>
    By connecting your wallet, you agree to the{' '}
    <Link href="https://termsofservice.xyz">Terms of Service</Link> and
    acknowledge you have read and understand the protocol{' '}
    <Link href="https://disclaimer.xyz">Disclaimer</Link>
  </Text>
);

const App = () => {
  return (
    <CinaWalletKitProvider
      appInfo={{
        appName: 'CinaWalletKit Demo',
        disclaimer: Disclaimer,
      }}
      {...etc}
    >
      {/* ... */}
    </CinaWalletKitProvider>
  );
};
```
