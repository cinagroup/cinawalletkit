# Senarai Dompet Tersuai

## Menyesuaikan senarai dompet

Anda boleh mengimport dompet individu dari `'@cinagroup/CinaWalletKit/wallets'` bersama dengan fungsi `connectorsForWallets` untuk membina senarai dompet anda sendiri dengan penyambung yang diperlukan. Dengan cara ini anda mempunyai kawalan penuh ke atas dompet mana yang akan dipaparkan, dan dalam susunan apa.

Sebagai contoh, anda boleh memilih untuk hanya memaparkan Rainbow bersama dompet pemulihan generik.

```tsx
import { connectorsForWallets } from '@cinagroup/CinaWalletKit';
import {
  rainbowWallet,
  walletConnectWallet,
} from '@cinagroup/CinaWalletKit/wallets';

const connectors = connectorsForWallets(
  [
    {
      groupName: 'Recommended',
      wallets: [rainbowWallet, walletConnectWallet],
    },
  ],
  {
    appName: 'My CinaWalletKit App',
    projectId: 'YOUR_PROJECT_ID',
  }
);
```

Anda kemudian boleh memberikan penyambung anda kepada `createConfig` Wagmi.

```tsx
import { connectorsForWallets } from '@cinagroup/CinaWalletKit';
import { createConfig } from 'wagmi';

const connectors = connectorsForWallets(/* ... */);

const config = createConfig({
  connectors,
  {/* Wagmi config */}
});

const queryClient = new QueryClient();

const App = () => (
  <WagmiProvider config={config}>
    <QueryClientProvider client={queryClient}>
      <CinaWalletKitProvider {...etc}>
        {/* Your App */}
      </CinaWalletKitProvider>
    </QueryClientProvider>
  </WagmiProvider>
);
```

### Dompet Terbina Dalam

Dompet berikut disediakan menerusi objek `wallet`.

#### Dompet Generik

Dompet berikut difokuskan pada kaedah sambungan generik dan bukan pada aplikasi khusus. Oleh itu, disyorkan agar dompet ini sentiasa disertakan.

##### WalletConnect

Ini adalah pilihan dompet pemulihan yang direka untuk dompet berasaskan WalletConnect yang tidak disediakan oleh dompet lain dalam senarai.

**Disarankan agar dompet ini sentiasa disertakan dalam senarai untuk memastikan semua dompet berasaskan WalletConnect disokong.**

```tsx
import { walletConnectWallet } from '@cinagroup/CinaWalletKit/wallets';
```

##### Dompet Suntikan

Ini adalah pilihan dompet pemulihan yang direka untuk senario di mana `window.ethereum` wujud tetapi tidak disediakan oleh dompet lain dalam senarai. Dompet ini secara automatik akan menyembunyikan dirinya dari senarai apabila pemulihan tidak diperlukan atau jika tiada dompet suntikan yang tersedia.

**Disarankan agar dompet ini sentiasa disertakan dalam senarai untuk memastikan semua dompet suntikan disokong.**

```tsx
import { injectedWallet } from '@cinagroup/CinaWalletKit/wallets';
```

##### Dompet Selamat

Untuk dApps yang menyokong Aplikasi Selamat bagi membolehkan pengguna menyambung dengan peti besi atau multi-sig mereka dengan mudah, sertakan pilihan `safeWallet` untuk sambungan automatik. Pilihan Selamat hanya akan muncul dalam persekitaran pelayar Dompet Selamat.

```tsx
import { safeWallet } from '@cinagroup/CinaWalletKit/wallets';
```

#### Base (sebelumnya dikenali sebagai Dompet Coinbase)

Penyambung dompet ini menyokong Base, evolusi Dompet Pintar Coinbase. Ia menyediakan pengesahan tanpa batasan dan ciri-ciri tambahan seperti sub-akaun dan kebenaran perbelanjaan.

Pilihan konfigurasi sejajar dengan [rujukan SDK Base](https://docs.base.org/base-account/reference/core/createBaseAccount).

> `baseAccount` dan `coinbaseWallet` kekal tersedia sebagai alias yang serasi ke belakang.

```tsx
import { base } from '@cinagroup/CinaWalletKit/wallets';
import { base as baseChain } from 'wagmi/chains';

// Configure attribution (choose one approach)
base.preference = {
  attribution: { auto: true }, // Smart Wallet generates origin-based identifier
};

// Or use custom attribution
base.preference = {
  attribution: { dataSuffix: '0x1234567890abcdef' }, // 16-byte hex identifier
};

// Configure sub-accounts
base.subAccounts = {
  creation: 'on-connect', // or 'manual'
  defaultAccount: 'sub', // or 'universal'
  funding: 'spend-permissions', // or 'manual'
};

// Configure paymaster URLs for gasless transactions
base.paymasterUrls = {
  [baseChain.id]: 'https://api.developer.coinbase.com/rpc/v1/base/...',
};
```

#### Dompet Gemini

Penyambung dompet pintar ini membolehkan pengesahan tanpa gangguan melalui kunci laluan di keys.gemini.com - tiada muat turun, sambungan, atau kunci API diperlukan. Ia berfungsi terus keluar dari kotak.

Parameter `appName` diperlukan untuk mengenal pasti dApp anda, dan `appIcon` adalah pilihan untuk penjenamaan pengalaman sambungan anda.

```tsx
import { geminiWallet } from '@cinagroup/CinaWalletKit/wallets';

const gemini = geminiWallet({
  appName: 'My App',
  appIcon: 'https://my-app.com/logo.png', // Optional
});
```

### Contoh

#### Penyusunan

Berikut adalah beberapa contoh paparan dompet yang berbeza, dalam urutan yang berbeza.

Paparkan MetaMask bersama dompet pemulihan generik.

```tsx
import { connectorsForWallets } from '@cinagroup/CinaWalletKit';
import {
  metaMaskWallet,
  walletConnectWallet,
} from '@cinagroup/CinaWalletKit/wallets';

const connectors = connectorsForWallets(
  [
    {
      groupName: 'Recommended',
      wallets: [metaMaskWallet, walletConnectWallet],
    },
  ],
  { appName: 'CinaWalletKit App', projectId: 'YOUR_PROJECT_ID' },
);
```

Tunjukkan Rainbow, MetaMask dan Base bersama dengan dompet tahan jatuh generik.

```tsx
import { connectorsForWallets } from '@cinagroup/CinaWalletKit';
import {
  rainbowWallet,
  metaMaskWallet,
  base,
  walletConnectWallet,
} from '@cinagroup/CinaWalletKit/wallets';

const connectors = connectorsForWallets(
  [
    {
      groupName: 'Suggested',
      wallets: [
        rainbowWallet,
        metaMaskWallet,
        base,
        walletConnectWallet,
      ],
    },
  ],
  { appName: 'CinaWalletKit App', projectId: 'YOUR_PROJECT_ID' },
);
```

> Peringatan: Urutan tatasusunan `wallets` menentukan urutan dompet akan dipaparkan pada antara muka pengguna.

#### Banyak kumpulan

Anda boleh menggunakan kunci `groupName` untuk menamakan kumpulan dompet yang berbeza. Ini berguna jika anda mahu berkomunikasi kepada pengguna anda dompet mana yang anda cadangkan, serta kemungkinan dompet lain.

Saran Rainbow dan MetaMask, tetapi juga tawarkan Base bersama dengan dompet tahan jatuh generik.

```tsx
import { connectorsForWallets } from '@cinagroup/CinaWalletKit';
import {
  rainbowWallet,
  metaMaskWallet,
  base,
  walletConnectWallet,
} from '@cinagroup/CinaWalletKit/wallets';

const connectors = connectorsForWallets(
  [
    {
      groupName: 'Recommended',
      wallets: [rainbowWallet, metaMaskWallet],
    },
    {
      groupName: 'Others',
      wallets: [base, walletConnectWallet],
    },
  ],
  { appName: 'CinaWalletKit App', projectId: 'YOUR_PROJECT_ID' },
);
```
