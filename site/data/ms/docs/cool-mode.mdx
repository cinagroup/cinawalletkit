# Mod Sejuk

## Mengaktifkan mod sejuk 游깯游깯游깯

Anda boleh mengaktifkan mod sejuk melalui prop `coolMode` pada `CinaWalletKitProvider`.

```tsx
import { CinaWalletKitProvider } from '@cinagroup/CinaWalletKit';

const App = () => (
  <CinaWalletKitProvider coolMode {...etc}>
    {/* Your App */}
  </CinaWalletKitProvider>
);
```

### Tetapi apakah itu?

Letupan emoji apabila dompet dipilih dari senarai dompet 游깯游깯游깯. Ia menggunakan ikon dompet yang anda pilih.
